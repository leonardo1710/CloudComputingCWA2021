## 1. Prerequisites
* Installation Node.js
* Installation einer IDE (im Tutorial wird VSCode vorwendet)
* Installation GIT

## 2. Projekt Setup

### Projekt klonen

``` git
git clone https://github.com/leonardo1710/cloud-starter-app.git
```

### Projekt Dependencies installieren

```
npm i 
```

### Projekt ausführen
```
npm run docs:dev
```
